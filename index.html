Projects

NEAT

An analysis of the genetic algorithm, NeuroEvolution of Augmenting Topologies (NEAT) developed by Ken Stanley in 2002 at UT Austin We first test the validity of NEAT components, then compare its performance to Q-Learning. The algorithms were compared based on their ability to find optimum solutions to balancing a pole and driving a car up a mountain.



Pi-Visualizer

Pi-Vis will be part of an art installation featured at Burning Man. Written in Python, the program makes extensive use of Socket programming and command line scripting. Via ethernet connection, the program manages playlist synchronization between two Raspberry Pis, each connected to their respective individual video output. Developed to withstand harsh environments, minimize the probability for failure, and provide users with easy interface and deployment, the program is both:

Redundant
Pis can be swapped on the fly without disruption to playback, and
Reliable
Errors handled by the system during the approx. 12hr run time

Lonr
Full-stack web app

Web-chat app generates Markov models from corpora to simulate conversation with notable comedians. Uses Express and SocketIO to establish two-way communication between client and server. NLTK processes user messages which are fed as input to a modified Python library, Markovify, to generate a comedian's message.
